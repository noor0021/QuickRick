# QuickRick

QuickRick is a very simple React app for auto visibility.

Flow:

1. Enter mobile number.
2. Enter OTP.
3. Open the dashboard.

Dashboard:

- Full map with zoom in and zoom out
- Current location tracking
- Where to? search box
- Passenger signal button
- Driver side signal count that updates instantly

## Tech stack

- React
- JavaScript
- React-Leaflet
- OpenStreetMap
- Vite

## Run locally

```bash
npm install
npm run dev
```

## Notes

I kept the code in one React file and one CSS file so it stays easy to read and edit.
